# Control Plane Config Reference

All variables are read from environment via `control_plane.config.ControlPlaneConfig`.

## General
`CONTROL_PLANE_ENABLED`, `CONTROL_PLANE_STRICT_MODE`

## Regime
`REGIME_ENGINE_ENABLED`, `REGIME_LOOKBACK_BARS_M5`, `REGIME_LOOKBACK_BARS_M15`, `REGIME_LOOKBACK_BARS_H1`, `REGIME_MIN_CONFIDENCE`, `REGIME_UNCERTAIN_BEHAVIOR`

## Regime thresholds
`TREND_STRENGTH_THRESHOLD`, `ROTATION_THRESHOLD`, `COMPRESSION_THRESHOLD`, `EXPANSION_THRESHOLD`, `DEAD_ZONE_THRESHOLD`, `EVENT_CHAOS_THRESHOLD`

## Allocator
`PORTFOLIO_ALLOCATOR_ENABLED`, `ALLOC_MAX_NEW_TRADES_PER_CYCLE`, `ALLOC_MAX_USD_NET_EXPOSURE`, `ALLOC_MAX_SINGLE_MACRO_CLUSTER_RISK`, `ALLOC_MAX_CORRELATED_BUCKET_RISK`, `ALLOC_MAX_INSTRUMENT_DUPLICATION`, `ALLOC_MIN_PRIORITY_SCORE`, `ALLOC_ENABLE_RESIZING`, `ALLOC_ENABLE_BLOCKING`, `ALLOC_PRIORITY_EV_WEIGHT`, `ALLOC_PRIORITY_EXEC_WEIGHT`, `ALLOC_PRIORITY_EDGE_WEIGHT`, `ALLOC_PRIORITY_REGIME_WEIGHT`, `ALLOC_PRIORITY_EVENT_WEIGHT`

## Correlation
`CORRELATION_LOOKBACK_DAYS`, `CORRELATION_MIN_ABS_FOR_CLUSTER`, `CORRELATION_REFRESH_MINUTES`

## Event
`EVENT_ENGINE_ENABLED`, `EVENT_CALENDAR_ENABLED`, `EVENT_PRE_LOCKOUT_MINUTES`, `EVENT_POST_DIGESTION_MINUTES`, `EVENT_SPREAD_NORMALIZATION_WINDOW_MINUTES`, `EVENT_ALLOW_BREAKOUT_POST_RELEASE_ONLY`, `EVENT_BLOCK_MEAN_REVERSION_NEAR_HIGH_IMPACT`, `EVENT_BLOCK_NEW_POSITIONS_IF_CALENDAR_UNAVAILABLE`

## Execution
`EXECUTION_INTEL_ENABLED`, `EXECUTION_FILL_PROB_THRESHOLD`, `EXECUTION_MAX_EXPECTED_SLIPPAGE_BPS`, `EXECUTION_MAX_LATE_ENTRY_RISK`, `EXECUTION_MAX_ESCAPE_RISK`, `EXECUTION_REPRICE_ENABLED`, `EXECUTION_DEFAULT_MAX_RETRIES`, `EXECUTION_DEFAULT_CANCEL_AFTER_SECONDS`

## Tactics
`ORDER_TACTICS_ENABLED`, `TACTIC_USE_PASSIVE_LIMIT_FOR_ROTATION`, `TACTIC_USE_AGGRESSIVE_STOP_FOR_EXPANSION`, `TACTIC_ALLOW_STAGING`, `TACTIC_MAX_CLIPS`, `TACTIC_MIN_CLIP_SECONDS`, `TACTIC_FALLBACK_TO_MARKET_ALLOWED`
